# Oi!

Installs the `oi` CLI which remembers things for you.

**Install:**

```bash
npm install -g oi-app
```

**Then learn something**

```bash
oi learn how do I rebase?
```

**Then ask a question:**

```bash
oi how do I rebase my git branch?
```
